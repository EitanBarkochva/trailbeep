# 🧭 מורה דרך (TrailBeep)

מדריך טיולים שמתריע ברקע: מצפצף ומקריא כשמתקרבים למעיין, אתר היסטורי או נקודת תצפית —
ומציג הסבר קצר מויקיפדיה בעברית. כולל מפת ישראל עם המיקום שלך בזמן אמת.

**אפליקציית PWA** (HTML/CSS/JS בלבד) — רצה בכל פלטפורמה מאותו קוד, וניתנת להתקנה למסך הבית.

## 🤖 גרסת אנדרואיד (מעקב אמיתי ברקע)
בנוסף ל-PWA, יש גרסת **אפליקציית אנדרואיד native** (Capacitor) שעוקבת **גם כשהמסך כבוי** —
בדיוק כמו Waze. נבנית אוטומטית ב-GitHub Actions.

### הורדת ה-APK
1. כנסו ל-[לשונית Actions](https://github.com/EitanBarkochva/trailbeep/actions/workflows/android.yml)
2. בחרו את הריצה האחרונה (✓ ירוקה) → באזור **Artifacts** הורידו את `more-derech-apk`
3. חלצו את ה-zip → קבלתם קובץ `app-debug.apk`

### התקנה בטלפון
1. העבירו את ה-APK לטלפון (או הורידו אותו ישירות מהטלפון)
2. פתחו אותו → אשרו **"התקנה ממקור לא ידוע"** אם נשאלתם
3. בפתיחה הראשונה אשרו:
   - **מיקום → "כל הזמן" (Allow all the time)** — חובה למעקב ברקע
   - **התראות** → אישור
   - **ביטול אופטימיזציית סוללה** לאפליקציה (כדי שאנדרואיד לא יעצור אותה)

מעכשיו אפשר להפעיל "התחל מעקב", לנעול את המסך ולשים את הטלפון בכיס — וזה ימשיך לצפצף.

### בנייה ידנית (אופציונלי)
```bash
npm install
npm run build:web      # מעתיק נכסים ל-www/
npx cap add android    # יוצר את פרויקט האנדרואיד
node patch-android.js  # מוסיף הרשאות רקע
npx cap sync android
cd android && ./gradlew assembleDebug
```

## מה זה עושה
- 🗺️ מפה חיה (Leaflet + OpenStreetMap) עם המיקום שלך.
- 💧🏺🌲🏕️🥾🎯 מסמן מעיינות, אתרים היסטוריים, תצפיות, פיקניק/חניוני לילה, שבילים ואתרי תיירות.
- 🔔 צפצוף + רטט + **הקראה קולית** (עברית/אנגלית) + באנר כשמתקרבים.
- 🚗 **מצב נהיגה** — כרטיס גדול עם האתר הבא בדרך, חץ כיוון וכפתור ניווט.
- 📖 הסבר קצר מויקיפדיה + ניווט ב-**Waze / Google Maps** (זוכר העדפה) + שיתוף בוואטסאפ.
- 📜 **היסטוריה** של מקומות שעברת לידם, מונה יומי וייצוא ל-CSV.
- ⚙️ הגדרות: קטגוריות, מרחק התראה, טווח לפי מהירות, סינון "רק חדש", מצב לילה.

## ✅ צ'קליסט לבדיקה בשטח
1. פתחו את הקישור בטלפון ואשרו **הרשאת מיקום** ("בעת השימוש").
2. ודאו שהנקודה הכחולה על המפה במיקום הנכון (לחצו 🎯 למרכוז).
3. לחצו **"התחל מעקב"** — סטטוס ה-GPS למעלה אמור להיראות ירוק.
4. בהגדרות ⚙️ — בדקו שהמסך נשאר דולק ("השאר מסך דולק"), והפעילו צפצוף/הקראה.
5. סעו לכיוון מעיין/אתר מסומן — אמורים לקבל **צפצוף + הקראה + באנר** בכניסה לרדיוס.
6. הקישו על נקודה → ודאו שמופיע הסבר וכפתורי הניווט פותחים את Waze/Google.
7. נסו **מצב נהיגה** 🚗 — שהאתר הבא והחץ מתעדכנים תוך כדי נסיעה.
8. בסוף — פתחו 📜 ובדקו שהמקומות נרשמו, ונסו ייצוא ל-CSV.

> 💡 GPS דורש HTTPS — לכן בודקים דרך כתובת ה-github.io, לא דרך קובץ מקומי.

## ⚠️ לגבי "רקע"
דפדפן בנייד **לא** מאפשר מעקב מיקום אמיתי כשהמסך נעול / האפליקציה סגורה —
מגבלת מערכת (iOS/Android). המעקב פועל **כל עוד האפליקציה פתוחה על המסך**; האפשרות
"השאר מסך דולק" (Wake Lock) שומרת על המסך דולק לתרחיש הדשבורד ברכב.

```bash

```

## מקורות נתונים (חינמיים, ללא מפתח)
- OpenStreetMap — אריחי מפה ושאילתות Overpass לאתרים.
- ויקיפדיה עברית/אנגלית — הסברים קצרים (REST summary + geosearch).
- CARTO — אריחי מפה כהים למצב לילה.

נא לא להעמיס על Overpass — האפליקציה שואבת מחדש רק אחרי תזוזה משמעותית.

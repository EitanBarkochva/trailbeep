# 🧭 מורה דרך

מדריך טיולים שמתריע ברקע: מצפצף כשמתקרבים למעיין, אתר היסטורי או נקודת תצפית —
ומציג הסבר קצר מויקיפדיה בעברית. כולל מפת ישראל עם המיקום שלך בזמן אמת.

**אפליקציית PWA** (HTML/CSS/JS בלבד) — רצה בכל פלטפורמה מאותו קוד, וניתנת להתקנה למסך הבית.

## מה זה עושה
- 🗺️ מפה חיה (Leaflet + OpenStreetMap) עם המיקום שלך.
- 💧🏺🌲 מסמן מעיינות, אתרים היסטוריים, תצפיות ואתרי תיירות מ-OpenStreetMap.
- 🔔 צפצוף + רטט + באנר כשמתקרבים (רדיוס שניתן להגדרה).
- 📖 הסבר קצר מויקיפדיה עברית בלחיצה על מקום + כפתור ניווט.
- ⚙️ המשתמש בוחר קטגוריות, מרחק התראה, צליל/רטט.

## ⚠️ לגבי "רקע"
דפדפנים בנייד **לא** מאפשרים מעקב מיקום אמיתי כשהמסך נעול / האפליקציה סגורה —
זו מגבלת מערכת (iOS/Android), לא משהו שניתן לעקוף בקוד אתר.
לכן המעקב פועל **כל עוד האפליקציה פתוחה על המסך**. האפשרות "השאר מסך דולק"
(Wake Lock) שומרת על המסך דולק כדי שאפשר להניח את הטלפון על הדשבורד ברכב.

מעקב רקע אמיתי (מסך כבוי) ידרוש בעתיד גרסת אפליקציה native.

## הרצה מקומית
צריך לשרת ב-`http://localhost` (לא לפתוח קובץ ישירות) — GPS דורש הקשר מאובטח.

```bash
# כל שרת סטטי מתאים, למשל:
npx serve .
# או
python -m http.server 8000
```
ואז לפתוח `http://localhost:8000`.

## הרצה בטלפון
GPS דורש **HTTPS**. הדרך הקלה: לפרסם ב-GitHub Pages / Netlify ולפתוח בכתובת ה-https.
באנדרואיד אפשר אז "הוסף למסך הבית" וזה יתנהג כאפליקציה.

## מקורות נתונים (חינמיים, ללא מפתח)
- OpenStreetMap — אריחי מפה ושאילתות Overpass לאתרים.
- ויקיפדיה עברית — הסברים קצרים (REST summary + geosearch).

נא לא להעמיס על Overpass — האפליקציה שואבת מחדש רק אחרי תזוזה משמעותית.
